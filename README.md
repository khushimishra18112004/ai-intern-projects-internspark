\```

## How to Run

### Task 1: ML Classification Model
\```
jupyter notebook "Task-1/ML Classification Model.ipynb"
\```
Then run all cells (Cell → Run All) or use:
\```
jupyter nbconvert --to notebook --execute "Task-1/ML Classification Model.ipynb"
\```
